# Copy Direction — SaaSWriter Landing Page

## Tone Rules
- **Direct, not clever** — no puns, no wordplay. State the value.
- **Pain-first** — lead with what's broken before what's fixed
- **Specific** — use real SaaS terms: PLG, SLG, churn, pipeline, motion, ICP
- **Confident, not salesy** — no "revolutionary" or "game-changing"
- **Monospace for meta** — section eyebrows and footnotes use code-style phrasing (`// how it works`, `free_during_beta`)
